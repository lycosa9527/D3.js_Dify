üîß Supported Chart Types

Thinking Maps¬Æ (Educational Diagrams)

  • Bubble Map - Central topic with connected attributes
  • Circle Map - Outer boundary with central topic and perimeter context
  • Double Bubble Map - Two topics with shared and unique characteristics
  • Bridge Map - Analogical relationships with relating factors

Traditional Charts

  • Bar Chart - Vertical or horizontal bars
  • Line Chart - Connected data points
  • Pie Chart - Circular segments
  • Scatter Plot - Points on X-Y axes
  • Area Chart - Filled areas under lines
  • Heatmap - Color-coded grid
  • Tree Map - Nested rectangles
  • Network Graph - Connected nodes
